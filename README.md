# intelligent_resume_analyzer_hidevs
The resume screening system is a python-based application designed to automate the candidate evaluation process by analyzing resumes and matching them against predefined job requirements. The system extracts essential candidate information, calculates compatibility scores, and generates professional hiring recommendations to assist recruiters in making informed decisions.
This project aims to reduce manual screening effort, improve hiring efficiency, and provide data-driven insights during recruitment.
Key features:
1. Automated resume parsing
2. Intelligent candidate matching
3. JSON based data management
4. Professional report generation
5. Error handling and robustness
Setup guide:
1. Install requirements
2. Create project folder
3. Add python files
4. Create job description file
5. Create resume dataset
6. Run the project
